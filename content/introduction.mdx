cs2.sh is an API platform that provides Counter-Strike 2 price data through a REST API.
The system collects data from independent marketplaces every 3-5 minutes.
All data is refreshed as soon as it's available.

Items use `market_hash_name`. For variant (e.g. Doppler) documentation, see [Variant Items](/variant-items).

Supported Marketplaces:

BUFF, Youpin, CSFloat, Steam, Skinport, C5Game

**`ask`** = Lowest listing price (what you'd pay to buy now).
**`bid`** = Highest buy order (what you'd receive if selling now).
All prices in USD.

## Quick Start

<Steps>
<Step title="Create an account">

[Sign up](https://cs2.sh/signup) and subscribe to a [plan](/docs/pricing).

</Step>
<Step title="Get your API key">

Generate an API key from your [dashboard](https://cs2.sh/dashboard).

</Step>
<Step title="Make your first request">

All authenticated requests require `Accept-Encoding: gzip`.

<CodeGroup>

```bash cURL
curl -X GET "https://api.cs2.sh/v1/prices/latest" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Accept-Encoding: gzip"
```

```python Python
import requests

response = requests.get(
    "https://api.cs2.sh/v1/prices/latest",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Accept-Encoding": "gzip",
    },
    timeout=30,
)

response.raise_for_status()
print(response.json())
```

```js Node.js
async function main() {
  const response = await fetch("https://api.cs2.sh/v1/prices/latest", {
    method: "GET",
    headers: {
      Authorization: "Bearer YOUR_API_KEY",
      "Accept-Encoding": "gzip",
    },
  });

  if (!response.ok) {
    throw new Error(
      `Request failed: ${response.status} ${response.statusText}`,
    );
  }

  const data = await response.json();
  console.log(data);
}

main().catch(console.error);
```

```go Go
package main

import (
	"fmt"
	"io"
	"net/http"
	"time"
)

func main() {
	req, err := http.NewRequest(http.MethodGet, "https://api.cs2.sh/v1/prices/latest", nil)
	if err != nil {
		panic(err)
	}

	req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
	req.Header.Set("Accept-Encoding", "gzip")

	client := &http.Client{Timeout: 30 * time.Second}
	resp, err := client.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()

	body, err := io.ReadAll(resp.Body)
	if err != nil {
		panic(err)
	}

	fmt.Println(string(body))
}
```

## Endpoints

For detailed explanations of each endpoint, see [Endpoints](/endpoints).

| I need...                       | Endpoint                   | Plan       |
| ------------------------------- | -------------------------- | ---------- |
| Current prices (all items)      | `GET /v1/prices/latest`    | Developer+ |
| Current prices (specific items) | `POST /v1/prices/latest`   | Developer+ |
| Recent price history (OHLC)     | `POST /v1/prices/history`  | Scale+     |
| Long-term price history         | `POST /v1/archive/history` | Scale+     |
| CSFloat sales history           | `POST /v1/archive/csfloat` | Scale+     |
| Liquidity scores                | `GET\|POST /v1/liquidity`  | Scale+     |
| Service health                  | `GET /health`              | Free       |
